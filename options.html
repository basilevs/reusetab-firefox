Matching Patterns


Usage

Blacklist

To prevent a site from being reused, add an expression at the top:

https://site_name.com/(.*)

To prevent a page from being reused, add an expression at the top:

https://site_name.com/page/.*

Multidomain

To treat two or more sites as the same site, add an expression at the top: 

https://(?:site1.com|site2.com)/.*

Technical information

Each line of configuration above is a regular expression. Tab is reused if a regular expression matches both target and source URLs and all capture groups of both matches are equal. Tab is not reused, if only one of target or source URLs are matching an expression. In examples above, blacklisting is achieved by introducing a capture group with value, that would be different for different pages of the same site; multidomain expression, ignores a segment of URl during matching by using non-capturing group, which is not used during comparison. First matching expression terminates further search, so higher expression have higher priority.
